# Hybrid Architectures

> 按需组合层级编排、并行执行、人机协同等多种模式，并利用框架原生的 Human-in-the-Loop 中断和流式输出能力，构建适应复杂场景的定制化 Agent 系统。

## 概述

**Hybrid Architectures** 并非固定某一模式，而是根据具体任务需求，混合 [[Hierarchical Agents]]、[[Swarm Agents]]、并行处理、反射循环等多种模式。

## 组合策略

- **层级 + 并行**：顶层监督者分解任务，子任务并行执行
- **人机协同 + 反射**：关键节点人工干预，反思循环自动优化
- **Swarm + 层级**：群体协作中引入层级管理

## 相关模式

- [[Hierarchical Agents]]：层级编排
- [[Swarm Agents]]：群体协作
- [[Graph-based Agents]]：图工作流
- [[Reflection Pattern]]：反思循环
- [[Agent Memory]]：记忆共享

## 关键事实

- Hybrid Architectures 按需组合多种 Agent 模式
- 利用框架原生的 Human-in-the-Loop 中断和流式输出
- 适应复杂场景的定制化 Agent 系统
- 可组合层级、并行、人机协同等多种模式
