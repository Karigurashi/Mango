## 四、技术面综合研判与策略

### 4.1 大盘技术面总结

```
┌─────────────────────────────────────────┐
│           大盘技术面综合评估              │
├─────────────────────────────────────────┤
│  上证指数：震荡偏弱  ★★★☆☆            │

│  整体判断：结构性行情，存量博弈            │
└─────────────────────────────────────────┘
```

**核心判断：**
1. **大盘处于宽幅震荡格局**（上证3950-4100点），短期无趋势性行情
2. **高低切换正在进行**：高位科技（AI算力、光模块、存储）资金持续流出，低位板块（军工、医药、金融）资金流入
3. **量能是关键**：若两市成交额持续低于3万亿，则存量博弈特征明显，板块轮动加速
4. **中报窗口期**：7月中下旬业绩验证期，纯概念股面临"估值双杀"风险
