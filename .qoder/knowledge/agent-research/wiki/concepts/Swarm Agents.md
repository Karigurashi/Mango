# Swarm Agents

> 去中心化的多 Agent 群体协作模式，Agent 之间通过消息传递和任务广播协调，无中心控制节点。

## 概述

**Swarm Agents**（群体 Agent）是一种去中心化的多 Agent 协作模式，Agent 之间通过消息传递和任务广播协调，没有中心控制节点。

## 核心特性

- **去中心化**：无中心控制节点
- **消息传递**：Agent 间通过消息协调
- **任务广播**：任务可广播给多个 Agent
- **自组织**：Agent 自主决定参与任务
